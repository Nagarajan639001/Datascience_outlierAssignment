what is 1,5 rule in IQR?
1.5 rule is a rule of thrumb for identfying outliers in a data set
It states that a data point is an outlier,it is morethan 1.5 times th interauantile range(IQR)
above the Q3 or below the Q1

The five number summaried for the day & Night classes

          Minimum       Q1        Median     Q3     maximum
    Day     32          56          74.5    82.5       99
    Night   25.5        78          81      89         98
    
   Day :  To calcuate the IQR : Q3-Q1 =82.5-56 = 26.5
                      Lesser  : Q1-1.5*IQR = 56-(1.5*26.5)=56-39.75=16.25
                      Greater : Q3+1.5*IQR = 82.5+(1.5*26.5)=82.5+39.75=122.25
                      
  Night :  To calcuate the IQR : Q3-Q1 =89-78 = 11
                      Lesser  : Q1-1.5*IQR = 78-(1.5*11)=78-16.5 =61.5
                      Greater : Q3+1.5*IQR = 89+(1.5*11)=89.5+16.5=105.50
